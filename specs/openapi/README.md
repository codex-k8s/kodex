# OpenAPI

Каталог для OpenAPI-спецификаций gateway-слоя.

Внутренние доменные сервисы не создают собственные прямые OpenAPI-спецификации. HTTP-поверхность делится по gateway-направлениям: внешний пользовательский доступ, staff/admin доступ, внешние интеграции и webhook. Каждая спецификация должна покрывать осмысленную поверхность, а не становиться одним большим файлом на всю платформу.

| Файл | Поверхность | Назначение |
|---|---|---|
| Пока нет активных спецификаций | - | HTTP-контракты будут добавляться вместе с gateway-сервисами. |
