# Устаревшие материалы

Этот каталог содержит материалы, которые раньше находились в корне репозитория, но больше не являются основой новой версии платформы.

## Что перенесено

| Старый путь | Новый путь | Статус |
|---|---|---|
| `services.yaml` | `deprecated/services.yaml` | Старый полный инвентарь сервисов, версий, окружений и runtime-настроек. |
| `cmd/` | `deprecated/cmd/` | Устаревший код. |
| `deploy/` | `deprecated/deploy/` | Устаревшие Kubernetes-шаблоны и deploy-материалы. |
| `libs/` | `deprecated/libs/` | Устаревшие общие библиотеки. |
| `proto/` | `deprecated/proto/` | Устаревшие транспортные контракты. |
| `services/` | `deprecated/services/` | Устаревшие сервисы. |
| `tools/` | `deprecated/tools/` | Устаревшие инструменты. |
| `docs/architecture/` | `deprecated/docs/architecture/` | Историческая архитектурная документация. |
| `docs/delivery/` | `deprecated/docs/delivery/` | Исторические delivery-планы, спринты и traceability. |
| `docs/ops/` | `deprecated/docs/ops/` | Историческая ops-документация. |
| `docs/product/` | `deprecated/docs/product/` | Историческая продуктовая документация. |
| `docs/research/` | `deprecated/docs/research/` | Исторические исследовательские материалы. |
| `docs/deprecated/` | `deprecated/docs/deprecated/` | Старый архив внутри `docs/**`. |

## Правила использования

- `deprecated/**` не является источником правды для новой реализации.
- Новые сервисы, контракты, deploy и библиотеки пишутся с нуля.
- Новые документы создаются по `docs/templates/**` и согласованной канонике `refactoring/**`.
- Старые материалы можно читать только как справку, чтобы понять исторический контекст или найти идею, которую нужно заново переоформить.
- Ссылки внутри `deprecated/**` не поддерживаются и не исправляются без отдельной задачи.
