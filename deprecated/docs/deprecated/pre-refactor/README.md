# Deprecated Pre-Refactor Docs

Этот каталог предназначен для документов старой модели платформы, которые будут вытесняться новой каноникой по программе рефакторинга.

Правила:
- сюда перемещаются только документы, которые уже заменены новым source of truth;
- перенос выполняется в том же PR, где новая каноника переносится из `refactoring/**` в `docs/**`;
- внутри archived документа должна сохраняться возможность понять, чем он был заменён.
