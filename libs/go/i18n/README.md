# i18n

## Назначение

`libs/go/i18n` задаёт общий backend runtime локализации системных текстов платформы.

Библиотека используется только для системных message id: названия действий, безопасные пользовательские сообщения, уведомления и ошибки, которые должны отображаться на локали пользователя. Пользовательские данные вроде названия проекта, описания задачи или текста комментария хранятся в одной введённой пользователем локали и не проходят через i18n runtime.

## Основа

Runtime построен на `github.com/nicksnyder/go-i18n/v2/i18n`.
