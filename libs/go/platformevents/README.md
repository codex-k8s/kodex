# Platform events

## Назначение

`libs/go/platformevents/**` содержит сгенерированные Go-константы типов доменных событий, типов агрегатов и общие payload-модели из AsyncAPI.

Файлы `*.gen.go` не правятся вручную. После изменения `specs/asyncapi/*.v*.yaml` выполнить:

```bash
make gen-asyncapi-event-contracts
```

Доменные сервисы могут использовать эти пакеты как общий контракт событий. Собственные доменные enum остаются внутри сервиса-владельца; межсервисные классификаторы синхронного API берутся из protobuf/gRPC, а асинхронные события — отсюда.
