# Карта Issue

## Назначение

Раздел хранит связи между GitHub Issue, PR, документами и волнами.

Карта намеренно разбита на несколько файлов, чтобы агенты, работающие над разными доменами, не конфликтовали в одном общем `issue_map.md`.

## Структура

| Путь | Назначение |
|---|---|
| `index.md` | Лёгкий индекс и сводка без длинных таблиц. |
| `domains/*.md` | Постоянные карты по доменам. |
| `waves/*.md` | Временные карты по волнам. |

## Правила обновления

- Доменный PR правит только файл своего домена и файл активной волны.
- Корневой `index.md` обновляется редко, только если добавлен новый домен или новая волна.
- После завершения волны её файл остаётся историческим снимком.
- В доменном файле остаётся актуальная долгоживущая связь `Issue/PR ↔ документы`.
- Шаблон `docs/templates/issue_map.md` применяется к каждому доменному или волновому файлу отдельно.
