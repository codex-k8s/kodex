# Оркестрация агентов

## Назначение

Домен описывает `agent-manager`, разбор пользовательских запросов, запуск ролевых агентов, управление flow, правила расписания, триггеры, продолжение сессий, машину приёмки, создание follow-up задач и артефактов, а также канонические flow, профили ролей и шаблоны промптов в БД.

## Что входит

- `agent-manager` как центральный управляющий агент;
- разбор пользовательских запросов;
- запуск ролевых агентов;
- управление flow;
- правила расписания и триггеры;
- продолжение сессий;
- машина приёмки;
- правила создания follow-up задач и артефактов;
- канонические flow, профили ролей и шаблоны промптов в БД;
- установочные фикстуры flow, ролей и шаблонов промптов в репозитории платформы.
