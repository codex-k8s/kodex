# Рабочие сущности провайдера

## Назначение

Домен описывает работу с provider-native сущностями: `Issue`, `PR/MR`, комментариями, mentions, relationships, ветками, тегами и полями провайдера.

Платформа не заменяет GitHub/GitLab как источник рабочих артефактов, а управляет orchestration, policy, audit, acceptance и projections вокруг них.
