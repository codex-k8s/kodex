# Биллинг и учёт затрат

## Назначение

Домен описывает записи затрат, распределение затрат по организациям, проектам и другим контурам, использование внешних провайдеров и runtime, основу для счетов, задел под платёжных провайдеров и экономику каталога пакетов.

## Что входит

- записи затрат;
- распределение по организациям, проектам и другим контурам;
- использование внешних провайдеров и runtime;
- основа для счёта;
- задел под платёжных провайдеров;
- экономика каталога пакетов.
