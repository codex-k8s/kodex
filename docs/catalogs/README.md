# Каталоги платформы

## Назначение

`docs/catalogs/**` описывает каталоги платформы: плагины, пакеты руководящей документации, роли и шаблоны. Эти каталоги не являются документацией конкретного проекта или домена.

## Разделы

| Раздел | Назначение |
|---|---|
| `package-store/` | Правила магазина пакетов как устанавливаемого пакета и источника доступных пакетов. |
| `plugins/` | Каталог плагинов и правила публикации/проверки. |
| `guidance-packages/` | Каталог пакетов руководящей документации для агентов. |
| `prompt-roles/` | Каталог встроенных и пользовательских ролей, процессов `flow` и шаблонов промптов. |

## Правило выноса

Каждый каталог должен проектироваться так, чтобы позже его можно было хранить в отдельном репозитории или подключать как внешний источник.

## Внешние источники

- `github.com/codex-k8s/kodex-package-store` — приватный пакет авторского магазина пакетов, необязательный submodule `external-docs/private/package-store`.
- `github.com/codex-k8s/kodex-platform-site` — приватный пакет сайта и пользовательской документации, необязательный submodule `external-docs/private/platform-site`.
- Правила работы с submodule описаны в `external-docs/AGENTS.md`.
