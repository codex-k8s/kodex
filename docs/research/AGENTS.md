# Инструкции для агентов в `docs/research/**`

## Назначение

`docs/research/**` хранит исходные материалы, исследования, заметки и сравнительный анализ, которые помогают принимать решения, но не становятся каноникой автоматически.

## Перед изменениями

- Читать `docs/AGENTS.md`.
- Читать `docs/research/README.md`.
- Читать `docs/templates/index.md`, если исследование оформляется как отдельный документ по шаблону.

## Правила содержания

- Исследование должно явно отделять факты, выводы и открытые вопросы.
- Если вывод принят как решение, его нужно перенести в `docs/platform/**`, `docs/domains/**` или `docs/catalogs/**`.
- Нельзя ссылаться на исследование как на источник обязательного поведения платформы, пока решение не перенесено в каноническую зону.
- Старые материалы из `deprecated/**` можно использовать только как справочный материал, если пользователь явно указал на них или они нужны для исторического контекста.
