# Исследовательские материалы

## Назначение

`docs/research/**` предназначен для исходных материалов, исследований и справочных заметок, которые помогают принимать решения.

Материал из этого раздела не становится каноникой автоматически. Чтобы использовать его в продукте или архитектуре, нужно переоформить выводы в `docs/platform/**` или `docs/domains/**`.
