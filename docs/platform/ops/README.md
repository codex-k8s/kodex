# Эксплуатация платформы

## Назначение

Раздел хранит сквозные эксплуатационные правила платформы. Доменная эксплуатационная документация размещается внутри доменных пакетов.

## Планируемые документы и шаблоны

| Документ | Шаблон | Смысл |
|---|---|---|
| Runbook | `runbook.md` | Описывает эксплуатационные действия и процедуры. |
| Мониторинг | `monitoring.md` | Фиксирует метрики, панели и наблюдаемость. |
| Уведомления и алерты | `alerts.md` | Описывает правила уведомлений и реакции. |
| SLO | `slo.md` | Фиксирует целевые уровни сервиса. |
| План отката | `rollback_plan.md` | Описывает безопасный откат изменений. |
| План релиза | `release_plan.md` | Описывает подготовку и выпуск релиза. |
| Заметки релиза | `release_notes.md` | Описывает состав выпуска. |
| Разбор инцидента | `incident_postmortem.md` | Фиксирует причины и выводы после инцидента. |
