# Продуктовая документация платформы

## Назначение

Раздел хранит сквозные продуктовые документы `kodex`: проблему, целевую модель, ограничения, MVP-рамки, персоны и нефункциональные требования на уровне всей платформы.

## Планируемые документы и шаблоны

| Документ | Шаблон | Смысл |
|---|---|---|
| Устав проекта | `project_charter.md` | Фиксирует цель платформы, владельца, рамки и способ принятия решений. |
| Проблема | `problem.md` | Описывает боль, которую решает платформа. |
| Краткое описание | `brief.md` | Даёт компактную модель продукта для быстрого входа в контекст. |
| Ограничения | `constraints.md` | Фиксирует неподвижные рамки и запреты. |
| Персоны | `personas.md` | Описывает пользователей, операторов и организации. |
| Рамки MVP | `scope_mvp.md` | Отделяет первую версию от enterprise-заделов. |
| Требования | `prd.md` | Описывает функциональные требования платформы. |
| Нефункциональные требования | `nfr.md` | Описывает безопасность, аудит, устойчивость и производительность. |
| Метрики успеха | `success_metrics.md` | Показывает, как понять, что цель достигнута. |
