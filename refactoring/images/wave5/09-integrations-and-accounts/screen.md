# Внешние аккаунты, интеграции и MCP

Статус: `reviewing`

## Назначение
Экран настройки внешних аккаунтов, provider-интеграций, MCP-серверов, лимитов и проблем авторизации.

## Главные сущности
- внешний аккаунт
- provider integration
- MCP-сервер
- account policy
- role binding
- rate limit
- auth issue

## Основные блоки
- верхняя лента summary-card по аккаунтам, интеграциям, проблемам авторизации и лимитам
- список категорий интеграций
- центральная таблица аккаунтов выбранного provider
- блок привязки аккаунтов к ролям и проектам
- блок MCP-серверов
- правая панель здоровья интеграций, последних auth issues и лимитов

## Ключевые действия
- добавить аккаунт
- проверить авторизацию
- открыть политику MCP
- настроить аккаунт
- настроить MCP-сервер
- посмотреть ограничения и квоты

## Источники данных
- платформа: provider-hub, account registry, MCP policy, rate-limit projection, auth issues
- внешние системы: GitHub, GitLab, OpenAI / Codex, каналы уведомлений

## Что намеренно не показывается
- сохранение секретов как читаемого контента экрана
- чат и голос как основной режим
- хаотичное смешение provider settings, runtime и release-диагностики
